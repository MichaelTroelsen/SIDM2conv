HOW TO RUN
================================================================================

Installation:
  1. Install PyQt6:
     pip install PyQt6

  2. Navigate to SIDM2 directory:
     cd ./SIDM2

Launching:
  Option 1 - Direct launch:
    python sf2_viewer_gui.py

  Option 2 - Using launcher (auto-installs dependencies):
    python launch_sf2_viewer.py

Usage:
  1. Launch the viewer
  2. Drag and drop an SF2 file onto the window, OR
  3. Click "Browse..." button to select a file
  4. View file information across 4 tabs:
     - Overview: Validation summary and file info
     - Header Blocks: All SF2 blocks with hex dump
     - Tables: View any table in grid format
     - Memory Map: Visual memory organization
