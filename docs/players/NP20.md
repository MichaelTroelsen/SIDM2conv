# NewPlayer 20 (NP20.G4) — SID → SF2 support

**Player:** NewPlayer 20 (the G4 variant)
**Registry key:** `np20`
**Driver:** `sf2driver_np20_00.prg`
**Accuracy:** **70–90%** (partial)
**Corpus:** NP20.G4 files in the broader SID set

NP20 is the predecessor family to Laxity's NP21. Support is **partial** — structure and notes convert, but fidelity is lower than the native Laxity path and varies per file.

---

## player-id strings → NP20
`NewPlayer_20` · `NewPlayer_20.G4` · `NP20`

---

## Status
- Notes/sequences extract; timbre/modulation fidelity is approximate (70–90%).
- Not the focus of recent work — the production path is native Laxity NP21 (see [LAXITY.md](LAXITY.md)).
- Some DRAX / NP21-fork files (Colorama/Delicate/Dreams/Omniphunk) share an NP21-derived codebase; see `memory/drax-np21-cluster-re.md`.

## Convert
```bash
sid-to-sf2.bat input.sid out.sf2 --driver np20
```

For the accuracy matrix across all players, see `docs/reference/ACCURACY_MATRIX.md`.
