QUICK START LAUNCHERS (.bat files in root)
===========================================

[MAIN TOOLS - GUI & CONVERSION]

  sf2-viewer.bat [file.sf2]
    - Interactive GUI for viewing and analyzing SF2 files
    - Features: Track View, OrderList, Sequences, Visualization, Playback
    - Recent files menu, drag-and-drop support
    - Examples:
        sf2-viewer.bat
        sf2-viewer.bat "learnings/Laxity - Stinsen.sf2"

  sf2-export.bat <input.sf2> [output_dir]
    - Export SF2 data to human-readable text files
    - Exports: orderlist, tracks, sequences, instruments, tables
    - Examples:
        sf2-export.bat "file.sf2"
        sf2-export.bat "file.sf2" "output/export"

  sid-to-sf2.bat <input.sid> <output.sf2> [--driver laxity|driver11|np20]
    - Convert SID files to SF2 format
    - Laxity driver: 99.93% accuracy (recommended for Laxity SIDs)
    - Examples:
        sid-to-sf2.bat "input.sid" "output.sf2"
        sid-to-sf2.bat "input.sid" "output.sf2" --driver laxity

  sf2-to-sid.bat <input.sf2> <output.sid>
    - Convert SF2 files back to SID format
    - Roundtrip validation support
    - Examples:
        sf2-to-sid.bat "input.sf2" "output.sid"

[BATCH OPERATIONS]

  batch-convert.bat [--roundtrip]
    - Convert all SID files in SID/ directory
    - Generates both NP20 and Driver 11 versions
    - Optional roundtrip validation
    - Examples:
        batch-convert.bat
        batch-convert.bat --roundtrip

  batch-convert-laxity.bat
    - Convert all Laxity NewPlayer v21 SID files
    - 99.93% frame accuracy, 8.1 files/second
    - 100% success rate on 286 test files
    - Examples:
        batch-convert-laxity.bat

  pipeline.bat [--quick] [--export-only]
    - Complete 13-step pipeline with validation
    - Steps: conversion, extraction, analysis, packing, dumps, accuracy,
             WAV, hex, trace, info, disassembly, validation, MIDI
    - Examples:
        pipeline.bat
        pipeline.bat --quick

[TESTING & VALIDATION]

  test-converter.bat [-v]
    - Run converter unit tests
    - 86 tests + 153 subtests, 100% pass rate
    - Examples:
        test-converter.bat
        test-converter.bat -v

  test-roundtrip.bat <input.sid>
    - Test SID→SF2→SID roundtrip conversion
    - Validates conversion accuracy
    - Examples:
        test-roundtrip.bat "SID/test.sid"

  validate-accuracy.bat <original.sid> <converted.sid>
    - Frame-by-frame accuracy validation
    - Detailed mismatch reporting
    - Examples:
        validate-accuracy.bat "SID/orig.sid" "output/conv.sid"

[MAINTENANCE & UTILITIES]

  cleanup.bat [--scan|--clean] [--force] [--update-inventory]
    - Clean temporary and experimental files
    - RULE 1: NEVER cleans git-tracked files
    - Creates backup list for every cleanup
    - Examples:
        cleanup.bat
        cleanup.bat --scan
        cleanup.bat --clean --force --update-inventory

  update-inventory.bat
    - Update docs/FILE_INVENTORY.md
    - Tracks complete repository structure
    - Examples:
        update-inventory.bat

  TOOLS.bat
    - Interactive menu launcher for all tools
    - Examples:
        TOOLS.bat

EXTERNAL TOOLS (tools/ directory)
==================================

  tools/siddump.exe
    - SID register dump tool (6502 emulation)
    - Use: tools/siddump.exe <input.sid> -t30 > output.dump

  tools/SIDdecompiler.exe
    - Emulation-based disassembler (player structure analysis)
    - Use: tools/SIDdecompiler.exe <input.sid>

  tools/player-id.exe
    - Player type identification
    - Use: tools/player-id.exe <input.sid>

  tools/SID2WAV.EXE
    - SID to WAV renderer
    - Use: tools/SID2WAV.EXE -t30 -16 <input.sid> <output.wav>

  tools/SIDwinder.exe
    - SID processor (disassembly, trace, player, relocate)
    - Use: tools/SIDwinder.exe <command> <input.sid>
