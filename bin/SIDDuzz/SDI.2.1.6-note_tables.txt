SDI 2.1.6 NOTETABLES FOR THE WAVEFORM PROGRAM (For all versions of SDI)
-------------------------------------------------------------------------------

##  = Value you put into the waveform program.
RES = Resulting note output.

The following values (##) can be entered in the note column of the 
waveform program, and will result in the note you can see under RES.
